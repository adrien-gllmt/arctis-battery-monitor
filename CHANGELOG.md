# Changelog

## [0.2.0] - 2026-03-09  

ABM now has a default icon in Windows menu.

### Fixes
- A bug that prevented the app showing up in notification tray and writing logs while it started properly on computer start up

## [0.1.0] - 2026-03-08

### Added
- Battery monitoring for SteelSeries Arctis headsets
- System tray icon reflecting battery level
- Notifications for connection, disconnection and multiple devices
- Language support (English, French)
- Start with Windows option
- Automatic update check via GitHub Releases, manually apply
